Details

  • Analyse exploratoire : Peu d'exigences formelles; l'objectif est de comprendre la distribution reelle des donnees.
  • Analyse confirmatoire : Importante pour les tests parametriques; des transformations ou des tests non-parametriques peuvent etre necessaires.
  • Analyse predictive : Critique pour certains modeles (regression lineaire); moins importante pour d'autres (arbres de decision).

Details

  • Analyse exploratoire : Peu d'exigences formelles; l'heteroscedasticite peut etre un resultat interessant en soi.

  • Analyse predictive : Peut affecter la precision et la generalisation des modeles; des transformations peuvent etre necessaires.

Valeurs aberrantes

Type d'analyseTraitement
Analyse exploratoireIdentification
Analyse confirmatoireTraitement
Analyse predictiveElimination ou modelisation

Details

  • Analyse exploratoire : Les valeurs aberrantes doivent etre identifiees et examinees, mais pas necessairement eliminees.
  • Analyse confirmatoire : Les valeurs aberrantes doivent etre traitees pour eviter de biaiser les resultats des tests.
  • Analyse predictive : Les valeurs aberrantes doivent etre eliminees ou modelisees specifiquement pour ameliorer la precision.

Details

  • Analyse exploratoire : Les patterns de donnees manquantes doivent etre identifies et analyses.

  • Analyse predictive : Les donnees manquantes necessitent des techniques d'imputation avancees ou une modelisation specifique.
